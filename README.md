## MotionLayout

Android Studio가 4.0으로 업데이트 되면서 Motion Editor가 추가되었습니다.  
해당 기능이 생김으로 인해서 MotionLayout을 적용하기가 더 쉬워졌고, 이 때다 싶어서 한 번 사용 해봤습니다.
