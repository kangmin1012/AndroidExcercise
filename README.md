## SpringAnimation

스프링이 달려 있듯이 탄성있는 애니메이션 작업입니다.

- 스프링 물리학으로 애니메이션화

- **감쇠비**와 **강성**옵션을 이용하여 스프링 세기 조절 가능

- 각 뷰마다 연쇄 애니메이션 적용 가능
